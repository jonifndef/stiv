# stiv - simple terminal image viewer

## Planned features:
- [ ] Open with path to image, open just this image
- [ ] Open with path to dir, open all images in gallery mode
- [ ] Open with no path, open all images in current dir in gallery mode
- [ ] Zoom, pan, rotate
- [ ] Zoom in gallery mode to increase, decrease thumbnail size
- [ ] Copy image or image name
### Possible future features:
- [ ] See subdirectories in tree-like browser in marigin window
