# stiv - simple terminal image viewer

## Planned features:
- [x] Open with path to image, open just this image
- [x] Open with path to dir, open all images in gallery mode
- [x] Open with no path, open all images in current dir in gallery mode
- [ ] Zoom, pan, rotate
- [ ] Zoom in gallery mode to increase, decrease thumbnail size
- [ ] Copy image or image name
- [x] When opened in a dir with multiple images, support goto next/previous while viewing single image
### Possible future features:
- [ ] See subdirectories in tree-like browser in marigin window
